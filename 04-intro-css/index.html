About CSS

CSS — Cascading Style Sheets — controls the look and feel of web pages. It defines colors, spacing, borders, fonts, and layout. Using an external stylesheet keeps your HTML clean and organized.

Three Ways to Apply CSS

  1. Inline: Inside a tag using the style attribute.
  2. Internal: Inside the <style> tag in the head.
  3. External: A separate .css file linked to multiple pages.
